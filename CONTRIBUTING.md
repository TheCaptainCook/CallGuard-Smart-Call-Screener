# Contributing to CallGuard

First off, thank you for considering contributing to CallGuard! It's people like you that make CallGuard such a great tool.

## 🌈 How Can I Contribute?

### Reporting Bugs
- Use the **GitHub Issue Tracker** to report bugs.
- Include a clear, descriptive title.
- Provide a step-by-step reproduction of the issue.
- Attach screenshots if applicable.

### Suggesting Enhancements
- Open a **Feature Request** on GitHub.
- Describe the motivation for the feature and how it would benefit users.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Issue that pull request!

## 📜 Code of Conduct
We are committed to providing a friendly, safe and welcoming environment for all, regardless of level of experience, gender, gender identity and expression, sexual orientation, disability, personal appearance, body size, race, ethnicity, age, religion, or nationality.

## 🛠️ Style Guide
- Follow standard Java/Android coding conventions.
- Use meaningful variable and method names.
- Ensure all new public methods have Javadoc comments.
